This repository contains solutions to algorithm problems from various online platforms such as LeetCode.

Known working versions: (`java --version`)

- `java 17.0.7 2023-04-18 LTS``

## Running a solution

1. Compile the java file `javac ./leetcode/MyFile.java`
2. Run the program `java leetcode.MyFile`

There is also a helper file `run.sh` that will compile and run the java solution.

Example: `./run.sh ./ik/SelectionSort.java`
